Single column form

This is the default form type

Notes

Forms are stacked and within panels

Modification of .form-control:

  • border-radius: 1px;
  • padding-left: 5px
  • padding-right: 5px

Inputs:

  • Mandatory fields have a * on label
  • Border of mandatory fields is rba(40, 40, 40)
  • input:focus with border-color: rgb(255, 145, 0)
  • No input with warning or success. Use only error fields

Put save and cancel buttons in a div with class .ba-form-buttons
